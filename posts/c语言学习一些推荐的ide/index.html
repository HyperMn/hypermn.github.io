C语言学习一些推荐的IDE
基于个人的使用体验，推荐一些C语言学习的IDE
2021-09-14 19:19 folder [杂文]   label C语言   IDE   schedule 4 min 9 s.

前言

工欲善其事，必先利其器。集成开发环境 (Integrated development environment)，简称IDE，是一种辅助程序开发人员编写程序的软件。在学习C语言的过程中我使用过很多不同的IDE，下面根据我的个人使用感受做一些C语言的IDE推荐。不过，无论大家使用怎样的IDE，多动手写代码才是最关键的，不必在IDE上的选择上过多纠结。

C语言推荐的IDE

下文中轻量、项目只是我个人对它们的理解。

win10系统：

轻量：适合编写简单程序

dev-cpp和code blocks是两款很有年头的软件了，用于写简单的程序非常方便，缺点也是功能比较少，界面不够现代化，胜在简单好用。

VSCode，微软的一款编辑器，在扩展的加持下功能强大，也是我的主力编辑器。不过使用它编写各种程序需要自己折腾，配置一下编译器、扩展等设置。

  • Dev-Cpp：Dev-C++ download | SourceForge.net
  • Code Blocks：Code::Blocks download | SourceForge.net
  • VSCode：可以参考win10系统使用VSCode配置C语言环境： https://zhuanlan.zhihu.com/p/298168673

项目：功能强大，用起来也比上面的IDE麻烦一些

CLion，JetBrains全家桶中的一员，功能非常非常强大，对学生免费，需要自行注册。

Visual Studio，被很多人称为宇宙第一IDE，很适合用来写C/C++/C#等程序。

  • CLion：CLion: A Cross-Platform IDE for C and C++ by JetBrains
  • VS2019：Visual Studio IDE: 面向开发人员的一体式编程工具 (microsoft.com)

Mac系统：

对MacOS不是很熟悉，所以只能推荐上面软件的mac版了。

轻量：VSCode

  • 可以参考mac下使用VSCode配置C语言环境：
  • mac vscode c++ 环境配置及异常的语法提示 - SegmentFault 思否

项目：CLion or XCode

  • CLion：对学生免费，需要自行注册CLion: A Cross-Platform IDE for C and C++ by JetBrains

  • XCode：AppStore下载

P.S: 上面的链接都是我随手搜索的，如果有问题的话大家可以在搜索引擎上搜索同样的内容多看几篇其他博客的教程。

在线编辑器

很多网站也提供了C语言的在线编辑器，在有网络但是没有配置开发环境的情况下也可以暂时使用在线编辑器。随便列举几个：

C 在线工具 | 菜鸟工具 (runoob.com)

c-clang在线开发工具，在线编译IDE_w3cschool

AC Editor (acwing.com)(需要注册，btw这个网站对想学算法的同学挺不错的)
